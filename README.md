# del
Recursively Shreds Files and Directories.

### Usage :-
```
del [path] <passes(optional)>
```

### Installation :-
1. `git clone https://github.com/0xB00B/del`
2. `$CXX -std=c++20 -O2 -s -static -flto del/del.cpp -o /usr/local/bin/del && rm -r del`

### Uninstallation :-
1. `rm /usr/local/bin/del`
